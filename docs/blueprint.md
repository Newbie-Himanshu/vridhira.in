# **App Name**: Vridhira Marketplace

## Core Features:

- Product Listing: We can list products with titles, prices, stock, categories, and images from admin panel.
- Shop: Displays products with a filtering capability.
- Secure Orders: Handles orders, including payment processing, status updates, and platform fee calculations.
- Admin Dashboard: Provides access to revenue control, and order management, secured via RBAC.
- Dynamic Fee Calculation: Generative AI tool suggests a suitable Platform Fee % based on current marketing data to maximize total revenue for artisans.
- Hybrid Product Search: Allows dynamic searching by product name, categories, and product descriptions.

## Style Guidelines:

- Primary color: Terracotta (#E07C54) to evoke a sense of Indian heritage and warmth.
- Background color: Warm Beige (#F5F5DC) to provide a neutral and inviting backdrop.
- Accent color: Deep Indigo (#4B0082) to add a touch of sophistication and contrast.
- Headline font: 'Playfair', a modern sans-serif with high contrast lines, imparting an elegant, fashionable feel; for longer text, use 'PT Sans' for body
- Code font: 'Source Code Pro' for displaying code snippets.
- Use icons inspired by Indian art and craft motifs.
- Classic, subtle design that reflects our work.
